## 🗂️ Planned Projects

### Project 1 · Hospital Readmissions Dashboard 
An interactive dashboard analyzing hospital readmission rates across service lines, payers, and patient demographics. Built to surface patterns that inform care transition and discharge planning strategies.

**Skills:** SQL, Python/pandas, Tableau or Power BI, hospital claims data

---

### Project 2 · HEDIS Diabetes Quality Measure Analysis
End-to-end HEDIS measure production for the Comprehensive Diabetes Care (CDC) measure set — identifying eligible members, applying technical specifications, and calculating numerator/denominator rates across a simulated health plan population.

**Skills:** HEDIS technical specs, ICD-10/CPT coding, SQL, claims + lab data

---

### Project 3 · Medicare Claims Cost & Utilization Analysis
Exploration of CMS Medicare public use files (or synthetic claims) to analyze spending patterns, high-cost conditions, geographic variation, and provider-level utilization trends.

**Skills:** CMS data (MBSF, MedPAR, or synthetic), Python, SQL, geospatial visualization

---

### Project 4 · 30-Day Readmission Predictive Risk Model
A machine learning model to predict 30-day all-cause hospital readmission risk at discharge, using clinical features (diagnoses, labs, vitals, LOS) and social risk factors. Includes model interpretability via SHAP values.

**Skills:** Python (scikit-learn, XGBoost), feature engineering, SHAP, ROC/AUC evaluation, EHR data

---

### Project 5 · Clinical NLP Documentation Integrity Analysis
Natural language processing pipeline applied to clinical notes to assess documentation completeness, identify diagnosis-documentation gaps (HCC risk adjustment relevance), and flag inconsistencies between free-text findings and coded diagnoses.

**Skills:** Python (spaCy, scispaCy, Hugging Face), NLP, MIMIC-III or synthetic notes, ICD coding logic
